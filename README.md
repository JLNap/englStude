# LinguaSprint MVP

Мини‑приложение для изучения английского:
- карточки для студентов;
- геймификация: XP, стрик, daily бонус, лиги и лидерборд;
- админка преподавателя (Supabase Auth + CRUD карточек);
- запись на урок после оплаты с генерацией ссылки Jitsi.

## 1. Настройка Supabase

1. Создай проект в Supabase.
2. В SQL Editor выполни файл `supabase-schema.sql`.
3. В Authentication включи Email/Password.
4. Скопируй `config.example.js` -> `config.js` и вставь:
   - `supabaseUrl`
   - `supabaseAnonKey`

## 2. Локальный запуск

Открой `index.html` в браузере.

## 3. Деплой на Vercel

1. Запушь репозиторий в GitHub.
2. Import Project в Vercel.
3. Framework preset: `Other` (статический сайт).
4. Deploy.

## 4. Как работать

- Студент: проходит карточки, оплачивает урок, получает Jitsi ссылку.
- Студент: получает XP/коины за правильные ответы, поднимается по лигам и попадает в рейтинг.
- Преподаватель: регистрирует аккаунт в админке, входит и управляет заданиями.
